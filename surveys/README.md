# Surveys

This directory holds repeatable compatibility surveys, VM captures, generated
comparison reports, and debugging notes.

```text
probes/original-compiler/  small byte-exact original compiler probes
action-runtime/            ACTION! RunTime Disk compatibility survey
toolkit/                   ACTION! Toolkit batch scripts and reports
tn/                        TOMS Navigator stability scripts and symbol dumps
stress/                    stress-fixture compatibility and MIR6502 surveys
```

For source material used by these surveys, see `corpora/`. For maintained
example programs, see `samples/`.
