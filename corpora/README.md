# Corpora

This directory holds archived inputs and byte-exact reference material. These
files are useful for compatibility work, but they are not the primary
user-facing sample set.

```text
action-runtime/  ACTION! RunTime Disk ATR and extracted files
toolkit/         ACTION! Toolkit ATR, extracted originals, and raw sidecars
tn/              TOMS Navigator ATR, extracted originals, and original COM
```

Survey scripts and generated reports live under `surveys/`.
