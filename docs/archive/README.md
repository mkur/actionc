# Documentation Archive

This archive keeps old implementation plans and investigation artifacts out of
the active documentation root while preserving their debugging value.

Files here are historical. They may mention old CLI names, obsolete failure
counts, or code paths that have since moved. Prefer the active docs in `../`
for current contracts and workflows.

## Layout

- `implementation-plans/general/` - broad optimization plans and roadmaps.
- `implementation-plans/mir6502/` - MIR6502 implementation and code-quality
  plans.
- `implementation-plans/semir-native/` - SemIR-native layer extraction plans.
- `implementation-plans/tac-nir/` - TAC, NIR, and source-span migration plans.
- `notes/` - old investigation notes.
- `reviews/` - old design/contract reviews.
- `snapshots/` - historical measurement snapshots.
